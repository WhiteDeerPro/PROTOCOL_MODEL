# TileLink LinkProtocol family

This directory reserves the LinkProtocol family boundary. It currently
contains no concrete builder or observer.

Planned conformance-level packages are `tl_ul/`, `tl_uh/`, and `tl_c/`.
Their common A/D channel vocabulary should be extracted only when the first
executable profile is implemented; TL-C coherence state will additionally
need SystemProtocol-visible components.
